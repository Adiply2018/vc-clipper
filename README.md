# ClipFinder — ゲーム動画セリフクリッパー

ゲームプレイ動画（VC入り）の音声をWhisperで文字起こしし、
特定のセリフ・キーワードが含まれる前後をクリップするデスクトップアプリ。

---

## 必要環境 (Windows)

- Python 3.10 以上
- ffmpeg（PATH に追加済みであること）
- GPU: NVIDIA CUDA対応推奨（なくてもCPUで動作）

## セットアップ

```
setup_and_run.bat
```

初回実行で仮想環境と依存パッケージのインストールが自動で行われる。

2回目以降:
```
run.bat
```

## 使い方

1. 動画ファイルをドラッグ&ドロップ（または「ファイルを選択」）
2. Whisperモデルを選択（auto推奨）
3. 「音声認識を開始」→ 文字起こしが完了するまで待つ
4. 「検索 & クリップ」タブでキーワードを入力 → 検索
5. 一致したセグメントの「クリップ」ボタンを押す
   - 「一致を全てクリップ」で一括出力も可能
6. 前後の秒数は左パネルで調整

## Whisperモデルの目安

| モデル | VRAM  | 速度 | 精度 |
|--------|-------|------|------|
| tiny   | ~1GB  | 速い | 低い |
| base   | ~1GB  | 速い | 普通 |
| small  | ~2GB  | 普通 | 普通 |
| medium | ~5GB  | 遅め | 高い |
| large  | ~10GB | 遅い | 高い |

CPU動作の場合は small または medium を推奨。

## 注意

- 長い動画（1時間超）はCPU認識だと30分以上かかることがある
- VCの複数話者は混合で認識される（話者分離は未対応）
- クリップはffmpegの `-c copy` で再エンコードなし（高速）
